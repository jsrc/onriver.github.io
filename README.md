# Benny's Blog

一个基于 Jekyll 的个人博客仓库，部署在 [onriver.top](https://onriver.top)。

这个站点最初从现成博客模板整理而来，当前版本已经收敛为适合继续自己维护的结构和内容。

## 本地预览

这个仓库保留了原始主题里的前端资源构建脚本：

```bash
npm install
grunt
```

如果本机装了 Jekyll，也可以直接本地预览站点：

```bash
jekyll serve
```

## 目录说明

- `_posts/`: 博文内容
- `_includes/`、`_layouts/`: 模板布局
- `css/`、`js/`、`less/`: 样式与脚本资源

## 说明

主题来自公开博客模板，并在此基础上做了定制和精简。
